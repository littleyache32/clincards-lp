# ClinCards — Landing Page

Landing page (lowticket) do kit de bolso ClinCards — 120 cards clínicos + 4 bônus, R$37.

Site estático (HTML/CSS puro, sem build), pronto pra deploy direto na Vercel.

## Estrutura

- `index.html` — página única com todas as seções (hero, problema, solução, prova, oferta, garantia, FAQ)
- Meta Pixel já instalado (PageView + InitiateCheckout nos CTAs)
- CTAs apontam para o checkout da Cakto

## Deploy

```bash
vercel --prod
```
